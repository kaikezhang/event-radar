# Market Regime & Amplification Factors

> 主人洞察：同一个事件在不同市场状态下的影响完全不同。市场超卖时一个好消息能引发大反弹；市场过热时一个坏消息就引发暴跌。

## 核心概念

事件的市场影响 = **事件本身冲击力 × 市场状态放大系数**

一个 "Trump 加关税" 的消息：
- 在市场超买 + VIX 低迷 + 杠杆率高的时候 → **暴跌 5-8%**（脆弱市场，雪崩效应）
- 在市场超卖 + 已经跌了 20% + 大家预期更差的时候 → **跌 1-2%**（利空出尽，预期已经很差了）

## 需要跟踪的 Regime 因子

### Tier 1: 核心市场状态（必须有）

| 因子 | 说明 | 数据源 | 更新频率 |
|------|------|--------|----------|
| **VIX** | 恐慌指数。VIX > 30 = 恐慌市场，事件冲击被放大 | Yahoo Finance / CBOE | 实时 |
| **SPY RSI(14)** | 相对强弱。RSI < 30 = 超卖（好消息反弹大），RSI > 70 = 超买（坏消息跌幅大） | 计算（需要价格数据） | 日频 |
| **SPY 距 52 周高/低** | 市场在什么位置？刚创新高 vs 跌了 20% | Yahoo Finance | 日频 |
| **SPY 20日均线 vs 50日均线** | 金叉/死叉 — 趋势方向 | 计算 | 日频 |
| **联邦基金利率** | 加息/降息周期 | FRED | 月频 |
| **10Y-2Y 利差** | 收益率曲线倒挂 = 衰退预期 | FRED | 日频 |

### Tier 2: 情绪与流动性（重要）

| 因子 | 说明 | 数据源 | 更新频率 |
|------|------|--------|----------|
| **Put/Call Ratio** | 看跌/看涨比。> 1.0 = 市场悲观（反转信号） | CBOE | 日频 |
| **AAII 散户情绪** | Bull% vs Bear%。极端悲观 = 反转信号 | AAII survey | 周频 |
| **Margin Debt** | 杠杆水平。高杠杆 + 坏消息 = 强制平仓雪崩 | FINRA | 月频 |
| **$MOVE Index** | 债券波动率。高 = 利率不确定性大 | ICE/Bloomberg | 日频 |
| **Credit Spreads** (HYG-IEF) | 信用利差扩大 = risk-off | 计算 | 日频 |
| **DXY (美元指数)** | 强美元 = 新兴市场/商品承压 | Yahoo Finance | 日频 |

### Tier 3: 板块轮动与极端信号（锦上添花）

| 因子 | 说明 | 数据源 |
|------|------|--------|
| **Advance/Decline Line** | 市场广度。大盘涨但 A/D 线背离 = 危险 | 计算 |
| **新高新低数** | 新高数骤降 = 隐性疲弱 | 计算 |
| **Sector Rotation** (XLK vs XLU 等) | 资金从科技流向公用事业 = risk-off | ETF 价格 |
| **Skew Index** | 尾部风险定价。高 skew = 大家买 OTM put 保护 | CBOE |
| **NAAIM Exposure** | 机构经理仓位水平 | NAAIM | 周频 |
| **Smart Money/Dumb Money Confidence** | 聪明钱 vs 散户的信心差 | SentimentTrader（付费） |

## 如何整合到 Pipeline

### Market Regime Score

将上述因子综合为一个 **regime score**（-100 到 +100）：

```
+100 = 极端超买/泡沫（坏消息冲击最大）
   0 = 中性
-100 = 极端超卖/恐慌（好消息反弹最大）
```

### 放大系数计算

| Regime Score | 好消息影响 | 坏消息影响 |
|-------------|-----------|-----------|
| +80 to +100（极度超买） | 0.5x（被忽略） | 2.0-3.0x（引发踩踏） |
| +40 to +80（偏高） | 0.7x | 1.5x |
| -40 to +40（中性） | 1.0x | 1.0x |
| -80 to -40（偏低） | 1.5x | 0.7x |
| -100 to -80（极度超卖） | 2.0-3.0x（引发轧空/反弹） | 0.5x（利空出尽） |

### 数据获取方案

**免费方案（v1）：**
- VIX / SPY / DXY → yfinance（每日更新，够用）
- 利率 / yield curve → FRED API（免费，需要 API key）
- RSI / MA 等 → 本地计算（只需要价格数据）

**专业方案（v2）：**
- 实时数据 → Polygon.io 或 Databento
- Put/Call Ratio → CBOE 网站 scrape
- AAII / NAAIM → 每周手动更新或 scrape

## 历史验证案例

### Case 1: 2025 年 10 月 Trump 关税暴跌
- **市场状态**: SPY RSI 72, 距 52w 高仅 -1%, VIX 14
- **事件**: Trump 宣布新一轮关税
- **结果**: SPY 单日 -3.8%
- **分析**: 超买市场 + 低波动 + 高杠杆 → 放大系数约 2x

### Case 2: 2025 年 3 月超卖后反弹
- **市场状态**: SPY RSI 28, 距 52w 高 -15%, VIX 32
- **事件**: Fed 暗示可能降息
- **结果**: SPY 单日 +2.8%
- **分析**: 超卖市场 + 高恐慌 + 空头拥挤 → 好消息放大系数约 2.5x

### Case 3: 中性市场的关税消息
- **市场状态**: SPY RSI 52, 距 52w 高 -5%, VIX 20
- **事件**: 类似关税消息
- **结果**: SPY -1.2%
- **分析**: 放大系数约 1.0x，符合基线
